Robbin' Hoodz

Robbin’ Hoodz is a 2D ‘endless runner’ browser game built with PhaserJS for my JavaScript capstone. While using Phaser was a new experience for me, this project involved many established skills and competencies of mine such as npm, webpack, Jest and ESLint to produce modular, testable and readable code. The final result of this project is, in my opinion, one of my best creations to date and was awarded Project of the Week on Career Karma.

TrainTrack

TrainTrack is a mobile time-tracking application I created as my capstone project for the Ruby on Rails module of the Microverse course. The app uses PostgreSQL as its database to store information about users, the transactions they make and the “groups” that those transactions belong to, all of which then supplies the front-end with the correct information to display, according to the Model-View-Controller pattern.

Gust

Gust is a web app that fetches and displays information about the current weather in any of the world's major cities. It was a small project I built during the JavaScript module on the Microverse course to demonstrate my mastery of API calls and asynchronous JS. The app simply takes the name of one of the world’s cities as its input and presents a card with basic information about the current weather in that city.

A Sailor, A Cook

A Sailor, A Cook is a website I created prior to enrolling with Microverse, which I built as part of a friend’s final-year university project. I led the design process and combined the client’s artistic direction with my understanding of Web Development and implemented the design effectively, adding extra features such as customised media players and a tab system. Finally, I managed the acquisition of a domain name and successfully deployed the site.

Icebreaker

Icebreaker is a lyrics-fetching web-scraper I created as my capstone project for Microverse’s Ruby module. I was tasked with creating a web scraper and, being a great lover of music, I set about creating a tool that could provide lyrics to a vast number of popular songs in a simple, quick and ad-free format. The finished product is an easy-to-use Terminal-based web-scraper that demonstrates quite nicely the affinity and comfort I feel working in Ruby.

Who I Am

If you’ve read this far, thank you for your interest in me and my work. My name’s Joe. I’m 24 years old and I’m originally from the U.K.

I discovered coding in 2019 and in the year since I’ve been slowly but surely building my skills up and working towards getting my first remote job as a developer. That journey really picked up with my induction into Microverse, a remote school for developers, in June 2020. I am now approaching the end of that course and will soon be on the job-hunt proper.

I’m a writer, a reader and a linguaphile. I speak conversational French and Spanish and basic German and I’ve lived and spent time in dozens of countries, on almost every continent. I’m a guitarist, a pianist and a music producer. In technology, I feel more drawn to the back-end of the stack; I’m interested in data, particularly machine learning, but I also love creating a good User Experience and, above all, a connectivity between people that inspires positive change for humanity and for the world.

My Stack

My core stack of technologies is as follows: HTML5 & CSS3, Ruby, SQL (Postgre/MySQL), Ruby on Rails, JavaScript with React and Redux. On the front-end, I often use Bootstrap, and I like to combine this with Sass for 'dry' and readable stylesheets. For testing Ruby/Rails and JS, I use RSpec and Jest respectively, and for code-linting I use Rubocop and ESLint. My personal language of preference is Ruby, and in the future I would love to learn Python and C, too.
